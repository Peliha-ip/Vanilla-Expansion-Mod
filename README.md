# Vanilla-Expansion-Mod
Here's the mod that compiles all my mods into one. Can make interactions that wouldn't be possible with them on their own. If you want to have more than one of my mods, use this instead.

This mod is a bundle of:
- Shellfish Reigns
- Borrowed Bonds
- Tusk Carnival
- The Cold Above
- Jurassic Pandemonium
- Profound Configuration
- End Colonialism
